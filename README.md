# 🤖 AI Agents Journey

Building AI agents from scratch — 7 weeks, 7 projects.

## Week 1 — Web Article Summarizer
An AI agent that fetches articles from URLs and summarizes
them automatically using HuggingFace BART model.

### Stack
- Python
- HuggingFace Inference API
- newspaper3k
- Agno framework (growing each week)

### Setup
1. Clone the repo
2. Create a virtual environment: `python -m venv venv`
3. Install dependencies: `pip install -r requirements.txt`
4. Copy `.env.example` to `.env` and add your HuggingFace token
5. Run: `python summarizer_agent.py`
